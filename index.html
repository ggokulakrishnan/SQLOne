SQLOne (the Acharya !!!)

(My learning with SQL Server ... )

The project contains series of SQL scripts (targeted for MS SQL Server engine) from Basics to Advanced level for the developer community

The SQL scripts were created for my personal learning, all the information in the Scripts are gathered from web (various blogs, sites, communities, usergroups, etc..). Hence sharing them back to the developer community as it will be useful for the developers who are planning to start their carrier in MS SQL Server

Scripts will be constantly updates with new features and releases from MS SQL Server ...

So lets get started and start learning MS SQL Server today :)

Happy Learning ...
